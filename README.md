# Python Network Security Labs

## Overview
This repository contains hands-on cybersecurity projects built while studying SOC operations, network security, and Python automation.

## Projects

### PortScan.py
A TCP SYN port scanner built using Scapy.

Features:
- TCP SYN scanning
- Open port detection
- DNS server identification

### HoneyScan.py
A defensive network deception tool inspired by honeypot concepts.

Features:
- Packet sniffing
- TCP response manipulation
- Honeypot port simulation

## Technologies
- Python
- Scapy
- Networking
- TCP/IP
- Packet Analysis

## Learning Outcomes
- TCP flags (SYN, SYN/ACK, RST/ACK)
- Network reconnaissance
- Defensive security concepts
- Packet crafting and analysis
